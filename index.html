RESPONSIVIDADE:responsividade refere-se à capacidade de um site, sistema ou aplicação se adaptar a diferentes dispositivos e tamanhos de tela, garantindo uma experiência consistente e funcional para o utilizador, independentemente do dispositivo utilizado (computador, telemóvel, tablet, etc.)

A responsividade permite que o layout do site, os elementos visuais e o conteúdo se ajustem automaticamente ao tamanho e à resolução do ecrã do utilizador.

A TAG "input" serve como um meio para introduzir dados num sistema computacional;

A TAG "label" representa uma legenda para um item em uma interface de usuário
